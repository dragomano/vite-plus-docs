# Удаление Vite+ {#removing-vite}

Используйте `vp implode`, чтобы удалить `vp` и все связанные с Vite+ данные с вашего компьютера.

## Обзор {#overview}

`vp implode` — это команда очистки для удаления установленного Vite+ и всех управляемых им данных. Используйте её, если больше не хотите, чтобы Vite+ управлял вашей средой выполнения, менеджером пакетов и связанным состоянием локальных инструментов.

::: info
Если вы решили отказаться от Vite+, пожалуйста, [поделитесь с нами своим отзывом](https://discord.gg/cAnsqHh5PX).
:::

## Использование {#usage}

```bash
vp implode
```

Чтобы пропустить запрос подтверждения, используйте:

```bash
vp implode --yes
```
